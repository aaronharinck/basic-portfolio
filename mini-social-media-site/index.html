Mini social media site

React, MobX

This was one of my first websites I made with react. The assignment was to create a mini social media website using states, react hooks, controlled components etc...

IncogniTOO is a social network with anonymous/incognito users, with the slogan 'connect with strangers'

People can signup quickly and pick a favorite color, or get a color randomly assigned. They can add posts, view other posts, leave comments and follow/unfollow other users.

The main purpose of this exercise was to learn about state management.
